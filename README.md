# PROJETO WEB SAÚDE
### Baixar o projeto:
```bash
git clone https://github.com/ifpi-picos/Web-Saude-Front-End.git
```
### Selecione a pasta app
```bash
cd app
```
### Instalar as dependências: 
```bash
npm install
```
### Executar o projeto:
```bash
npm run dev 
```
### Acessar sitema:
[http://localhost:3000/](http://localhost:5000/)
